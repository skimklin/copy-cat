## :rocket: 使用者指南

通过npm下载安装代码

```bash
$ npm install --save copy-cat
```

如果你是node环境

```js
var CopyCat = require('copy-cat');
```

如果你是webpack等环境

```js
import CopyCat from 'copy-cat';
```

如果你是浏览器环境

```html
<script src="node_modules/copy-cat/dist/index.aio.js"></script>
```
